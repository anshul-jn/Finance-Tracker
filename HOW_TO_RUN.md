## 🆘 Troubleshooting

### **Error: Java compiler not found**
- Install JDK (Java Development Kit)
- Add Java to your PATH

### **Error: sqlite-jdbc JAR not found**
- Make sure `lib/sqlite-jdbc-3.51.3.0.jar` exists
- Check the version number

### **Compilation errors**
- Check the error message above
- Make sure all .java files are in correct folders
- `src/com/expensetracker/` folders should exist
